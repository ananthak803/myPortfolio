# My Portfolio

This is my personal portfolio website built using React and Tailwind CSS.

## Tech Stack

* React (Vite)
* Tailwind CSS
* React Router
* Framer Motion
* GSAP

## Features

* Responsive design
* Animated hero section
* Navigation menu
* Sections: Home, About, Experience, Projects, Contact

## Installation

1. Clone the repo

```bash
git clone https://github.com/your-username/your-repo-name.git
```

2. Go to project folder

```bash
cd your-repo-name
```

3. Install dependencies

```bash
npm install
```

4. Run the project

```bash
npm run dev
```
